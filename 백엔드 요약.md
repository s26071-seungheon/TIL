# 백엔드

## 백엔드
 ### 백엔드는 사용자 직접 보지 않는 쪽에 서비스가 동작하도록 하는 서버 쪽 프로그램이다.
 
## 백엔드 개발자가 하는 일
 ### 1. 사용자의 요청을 처리하는 서버 로직을 개발한다.
  #### ex)로그인, 회원가입, 게시글 작성, 결제 처리 등 서비스의 핵신 기능을 구현한다.
 ### 2. 데이터베이스를 관리한다.
  #### ex)사용자 정보, 게시글, 상품 정보 같은 데이터를 데이터베이스에 저장하고 조회하고
  ####    데이터 구조를 설계하고 성능을 최적화한다.
 ### 3. API를 만든다.
