# 1강
 ## public class HelloJava
  #### HelloJava.java 라는 파일을 만들었다고 이해
 ##  public static void main(String[] args)
  #### 프로그램의 실행 시점이 되는 부분이다.
  #### main은 지금은 시작점이라 생각하면 된다.
 ## System.out.println
  #### 콘솔 화면에 값을 출력하는 명령어이다.
  #### 자바는 대소문자를 구분하기 때문에 대소문자를 다르게하면 오류가 발생할 수 있다.
 ## {},;,들여쓰기
  #### {}블록을 사용해서 클래스의 시작과 끝을 나타낸다.
  #### ;는 문자을 구분하는 기호이다. 문장이 끝나면 세미콜론을 필수로 넣어줘야한다.
  #### 들여쓰기는 코드를 쉽게 구분하고 이해하도록 도와주는 좋은 관례이다.(보통 4번 스페이스 사용 or Tab)
