# 클라우드
 ## 역할
 - 인터넷으로 서버, 자원등을 빌려 서비스를 안정적으로 운영

 ## 하는 일
 - 서버 제공
   - 웹사이트, 앱을 인터넷에 띄울 수 있음
     - ex) 게임 서버, 쇼핑몰
 - 데이터 저장
   - ex) 사진, 파일, 영상
 - 서비스 운영
   - 트래픽 많아지면 자동으로 서버 확장
     - 트래픽 : 인터넷에서 데이터가 오가는 양
 
 ## 사용하는 도구
 - Google Cloud
 - Amazon Web Services
 - Microsoft Azure

 ## 사용하는 언어
 - Python
 - Java
 - JavaScript
 - Go
