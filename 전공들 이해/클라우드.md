# 클라우드
 ## 역할
 - 인터넷으로 서버, 자원등을 빌려 서비스를 안정적으로 운영

 ## 하는 일
 - 서버 제공
   - 웹사이트, 앱을 인터넷에 띄울 수 있음
     - ex) 게임 서버, 쇼핑몰
 - 데이터 저장
   - ex) 사진, 파일, 영상
 - 서비스 운영
   - 트래픽 많아지면 자동으로 서버 확장
     - 트래픽 : 인터넷에서 데이터가 오가는 양
 
 ## 사용하는 도구
 - Google Cloud
 - Amazon Web Services
 - Microsoft Azure
 - Docker

 ## 사용하는 언어
 - Python
 - Java
 - JavaScript
 - Go

 ## 클라우드와 백엔드
 - 배포환경 구성
   - 백엔드가 스프링으로 서버 같은 거 만들면  클라우드가 도커 이용하여 서버(AWS등)에 올리고 주소 연결
 - 데이터베이스 관리 
   - 백엔드가 데이터베이스를 MySql로 쓴다고 하면 클라우드가 클라우드용 DB서비스(AWS RDS등)를 세팅하고 매일 백업되게 해놈
 - 모니터링 로그
   - 클라우드가 서버 오류 확인하면 백엔드가 오류확인하고 고친다
