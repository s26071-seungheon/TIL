# 프론트엔드
 ## 역할
 - 사용자가 직접 보는 화면을 구현하고 웹사이트,앱과 상호작용 할 수 있게 함

 ## 하는 일
 - HTML로 구조 만들기
   - 웹페이지 전체구조 구현
   - 글, 사진, 버튼 배치
 - JavaScript로 동작 처리
   - 버튼 클릭하면 반응
   - 입력처리
   - 메뉴 열기, 닫기
 - CSS로 꾸미기
    - 글자 색 바꾸기
    - 배경색 넣기
    - 버튼 모양

 ## 사용하는 언어
 - HTML
 - CSS
 - JavaScript

 ## 사용하는 도구
 - React
 - Vue.js
 - Angular

 ## 프론트엔드와 백엔드 호환방법
 - API라는 통로로 연결한다
 - 프론트엔드가 백엔드에 요청
 - 백엔드가 프론트엔드에게 응답
 - 데이터 형식은 JSON 형식
 - 프론트엔드 : fetch(),axios
 - 백엔드 : Controller, RESTAPI
