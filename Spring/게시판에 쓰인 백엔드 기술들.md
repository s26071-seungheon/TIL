# Spring Framework
 - 자바 기반 웹 애플리케이션 개발 프레임워크
 - MVC 구조로 요청 처리
 - DI/IoC지원
    - DI : 필요한 객체 외부에서 주입
    - IoC : 프로그램 흐름을 프레임어크에게 맡김
   
# Sping Boot
 - Sprng 설정 최소화하고, 빠르게 실행 가능한 환경 제공

# Spring MVC
 - Model,View,Controller 구조로 요청 처리

# JPA (Java Persistence API)
 - DB와 객체(Entity)를 매핑, CRUD 자동 제공

# MySQL
 - 게시판 데이터를 저장하는 DBMS
