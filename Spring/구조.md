
# Controller
 ## HomeController
  ### 웹에서 /주소로 들어오면 resource안에 있는 index HTML을 보여주는 코드

 ## BoardController
  ### 웹에서 게시글 생성, 조회, 수정, 삭제(CRUD)를 처리하는 코드

# Entity
 ## BaseTimeEntity
  ### 모든 엔티티 생성 시간, 수정시간 자동관리 해주는 부모 클래스

 ## BoardTimeEntity
  ### DB와 게시글 데이터 매핑 + DTO를 엔티티로 변환 + 수정기능

# DTO(Data Transfer Object)
 ## BoardDTO
  ### 객체의 데이터를 필요있는 것만 DTO로 옮기는 코드

# repository
 ## BoardRepository
  ### 게시판에선 조회수 증가 기능을 담당하고 있는 코드

# service
 ## BoardService
  ### 게시판에선 기능(저장, 조회, 수정, 삭제)을 처리하는 중간 관리자 역할
