# 배열(Array)
 ### students=배열명
  ### int[ ] students은 배열 변수 선언이다.
  ### students = new int[숫자] = 배열을 생성한다.
  ### students.length 배열의 길이 제공
  ### students = new int[]{90,80,70,60,50} 처럼 배열생성과 초기화를 같이할 땐 대괄호 안에 숫자를 빼준다.
  ### int[] students = new int[]{90,80,70,60,50} 처럼 선언과 생성 초기화를 한번에 가능하다.
  ### int[] students = {90,80,70,60,50} 처럼 new int[] 생략이 가능하다.

 ## 배열의 초기화
 ### 자바는 배열을 생성할 때 그 내부값을 자동으로 초기화한다. 숫자는 0, boolean은 false, String은 null

 ## 배열 참조값 보관
 ### 배열을 생성하면 배열의 크기만큼 메모리를 확보한다.
 #### ex) int형 5개는 4byte * 5 = 20byte를 확보한다.
 ### 배열을 생성하고 나면 자바는 메모리 어디간에 있는 이 배열에 접근 할 수 있는 참조값(주소)(x001)을 반환한다.
 #### 여기서 x001이라고 표현한 것이 참조값이다. (실제로 x001처럼 표현되는 것은 아니고 이해를 돕기위한 예시이다.)
 ### 배열 변수 선언한 곳인 int[ ] students에 생성된 배열의 참조값(x001)를 보관한다.
 ### int[ ] students변수는 new int[숫자]로 생성한 배열의 참조값을 가지고 있는 것이다.
#### 이 변수는 참조값을 가지고 있다. 이 참조값을 통해 배열을 참조할 수 있다. 쉽게 이야기해서 참조값을 통해 메모리에 있는 실제 배열에 접근하고 사용할 수 있다.
#### 배열을 생성하는 new int[5] 자체에는 아무런 이름이 없다. 단순히 int 형 변수 5개를 연속으로 만든 것이다. 따라서 생성된 배열에 접근하려면 참조값을 어딘가에 보관해야 한다. int[] students 변수에 참조값(x001)을 저장하면, 이 변수를 통해 배열에 접근할 수 있다.

## 2차원 배열
 ### int[ ][ ] arr = new int[2][3]와 같이 선언하고 생성한다.
 ### arr[1][2] = 1같이 숫자를 집어 넣는다.,행이 먼저고 열이 뒤이다.
### 이렇게도 가능하다.
        int[][] arr = new int[][]{
            {1,2,3},
            {4,5,6}
        };
### 배열 크기만 정해놓고 값을 반복문으로 넣을 수 있다.
    package Array;

    public class ArrayDi4 {
    public static void main(String[] args) {
    int[][] arr = new int[2][3];

        int com = 1;
        for (int i = 0; i < arr.length; i++) {
            for (int j = 0; j < arr[i].length; j++) {
                arr[i][j] = com++;
            }
        }
        for (int i = 0; i < arr.length; i++) {
            for (int j = 0; j < arr[i].length; j++) {
                System.out.print(arr[i][j] + " ");
            }
            System.out.println();
        }
    }
    }
