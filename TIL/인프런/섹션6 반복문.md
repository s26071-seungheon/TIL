# 34강
 ## 반복문
  #### 반복문은 코드를 반복해서 실행할 때 사용한다.

### 종류
#### while
#### do-while
#### for

# 35강
 ## while문
  #### while(조건식) { //코드 } 방식
  #### 조건식 확인하고 참이면 코드 블록 실행 거짓이면 벗어난다.

# 37강
 ## do while
  #### while문과 비슷하지만 조건에 상관없이 무조건 한 번은 코드를 실행한다.

# 38강
 ##break
  #### 반복문을 즉시 종료하고 나간다

 ## continue
  #### 반복문의 나머지 부분을 건너뛰고 다음 반복으로 진행하는데 사용한다.
