# 26강
 ## 조건문
   #### 특정 조건에 따라 코드를 실행하게 함
   #### if문, switch문이 있다.

 ## if문
   #### if문은 특정 조건이 참인지 확인하고, 그 조건이 참(ture)일 경우 특정 코드 블록을 실행한다.

 ## else문
  #### else문은 if문에서 만족하는 조건이 없을 때 실행하는 코드이다. 

# 27강
 ## else if문
 #### else if문은 앞선 if문의 조건이 거짓일 때 다음 조건을 검사한다. 만약 if문이 참이라면 else if문은 실행하지 않는다.
 #### 중복 검사를 피할 수 있다.
