# 26강
 ## 조건문
   #### 특정 조건에 따라 코드를 실행하게 함
   #### if문, switch문이 있다.

 ## if문
   #### if문은 특정 조건이 참인지 확인하고, 그 조건이 참(ture)일 경우 특정 코드 블록을 실행한다.

 ## else문
  #### else문은 if문에서 만족하는 조건이 없을 때 실행하는 코드이다. 

# 27강
 ## else if문
 #### else if문은 앞선 if문의 조건이 거짓일 때 다음 조건을 검사한다. 만약 if문이 참이라면 else if문은 실행하지 않는다.
 #### 참인 곳에서 딱 멈추기 떄문에 중복 검사를 피할 수 있다.

 # 28강
  ## if문과 else if문 심화
  #### if문에 여러 조건이 있다고 if-else로 묶어서 사용 할 수 있는 것은 아니다.
  #### 조건이 서로 영향을 주지 않고 각각 수행해야 할 경우는 else if문을 사용하지 않고 if문을 분리해서 사용한다.
  #### if문에서 한 줄 만 있을 땐 중괄호를 생략할 수 있다.
  #### 가독성과 유지보수성 때문에 명령어 한 개만 있을 경우에도 중괄호를 사용하는 것이 좋다.

 # 29강
  ## switch문
   #### switch문은 단순히 값이 같은지만 비교할 수 있다.
   #### 조건식의 결과 값이 어떤 case의 값과 이치하면 해당 case의 코드를 실행한다.
   #### break문은 현재 실행 중인 코드를 끝내고 switch문을 빠져나가게 하는 역할을 한다.
   #### 만약 break문이 없으면, 일치하는 case 이후의 모든 case코드들이 순서대로 실행된다.
   #### default는 조건식의 결과값이 모든 case의 값이 일치하지 않을 때 실행,default는 선택

  ## 상향 switch문
   #### : 대신 -> 를 사용
   #### 선택된 데이터를 반환할 수 있음
   
 # 30강
  ## 삼항 연산자
   #### 삼항 연산자는 항이 3개라는 뜻. 조건, 참_표현식, 거짓_표현식 이렇게 항이 3개
   #### 조건에 맞으면 참_표현식이 실행, 조건에 만족하지 않으면 거짓_표현식이 실행
   #### 단순한 표현식만 넣을 수 있음
